Instalação

Para instalar este app em seu pc e poder estudar suas funções siga os seguintes passos:

  • Clone o projeto do GitHub.
  • Abra a pasta do projeto com o Visual Studio Code.
  • No VS Code use o atalho de abrir o prompt de comando: ctrl + shift + '
  • Intale o node modules com o comando: npm install
  • Compile o projeto com o comando: ionic build
  • Copie a compilação para o projeto com o comando: ionic cap copy
  • Sincronize todos os plugins com o comando: ionic cap sync
  • Instale o Android Studio mais recente que tiver disponível.
  • Abra o projeto no Android Studio com o comando: ionic cap open android
  • Conecte seu Smatphone com um cabo de dados no pc, libere o modo desenvolvedor no seu smartphone e deixe a conexão no modo ancoragem USB
  • Instale para teste em tempo real no seu smartphone com o comando: ionic cap run android -l --external
  • Seu Smartphone devera aparecer no prompt de comando, selecione ele e digite enter.
  • O Ionic deverá oferecer alguns servidores (IPs) pra você rodar o app em modo Live Reload (tempo real) enquanto programa, escolha algum e espere compilar no seu Smartphone.
  • Se tudo der certo o app abrirá no seu smartphone e tudo que você mudar no código deverá mudar tmb no app em tempo real.
